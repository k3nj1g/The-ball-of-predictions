# Мистический шар

Одностраничное веб-приложение с анимированным энергетическим шаром, персональными мистическими предсказаниями и шарингом.

## Возможности

- Анимированный canvas-шар с режимами `idle` / `thinking` / `result`.
- Поле имени с валидацией (очистка спецсимволов, лимит до 32 символов).
- Задержка 1.5–3.5 сек для эффекта «шар размышляет».
- Генератор предсказаний в стиле мистики + тёмной иронии.
- Поддержка Web Share API, fallback на копирование текста, ссылки для Telegram / VK / X.
- Базовая доступность: `aria-label`, focus-visible, мобильная адаптация, `prefers-reduced-motion`.

## Локальный запуск

Так как это статический проект без сборки, достаточно любого HTTP-сервера.

### Вариант с Python

```bash
python3 -m http.server 4173
```

Откройте в браузере: `http://localhost:4173`

## Структура

- `index.html` — разметка страницы и UI-блоков.
- `styles.css` — стили, адаптивность, состояние элементов.
- `app.js` — состояние приложения, анимация шара, обработчики и шаринг.
- `predictions.js` — наборы шаблонов предсказаний и генератор.

## Как менять тексты предсказаний

1. Откройте `predictions.js`.
2. В объекте `toneSets` отредактируйте массивы:
   - `omens`
   - `warnings`
   - `taglines`
3. При желании добавьте новый тон (по аналогии) и подключите его в `generatePrediction`.

> Сейчас реализованы три тона: `darkOmen`, `sardonic`, `arcaneAdvice`.

## Лицензии и ассеты

Проект не использует стоковые видео/изображения и водяные знаки. Визуал шара полностью отрисовывается на Canvas.
